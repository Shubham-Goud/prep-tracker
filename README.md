# 🚀 PrepTracker – 100 Days Placement Preparation Platform

🔗 Live App: https://prep-tracker-g6hs.onrender.com

---

## 📌 Overview

PrepTracker is a **full-stack web application** designed to help students prepare for placements with a **structured 100-day roadmap** covering:

* 📘 Data Structures & Algorithms (DSA)
* 🐍 Python
* 🗄️ SQL

It provides **daily tasks, problem-solving practice, progress tracking, and streak management** — all in one platform.

---

## ✨ Features

* 📅 **100-Day Structured Plan**
* 🔥 **Daily LeetCode Problems (Clickable)**
* 📊 **Progress Tracking System**
* 🔥 **Streak Tracking**
* 🗄️ **SQL Practice Section (100 Days)**
* ✅ **Mark Problems as Solved**
* 🎯 **Clean UI for Daily Learning**
* 🌍 **Live Deployed Application**

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS (Jinja Templates)
* **Backend:** Flask (Python)
* **Database:** SQLite
* **Deployment:** Render
* **Version Control:** Git & GitHub

## 🚀 How to Run Locally

```bash
git clone https://github.com/Shubham-Goud/prep-tracker.git
cd prep-tracker

pip install -r requirements.txt
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```
